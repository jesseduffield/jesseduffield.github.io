I'm Jesse Duffield, a software engineer by trade and currently Co-founder/CEO of [Subble](https://www.subble.com/), based in Melbourne, Australia.

I'm best known for creating [Lazygit](https://github.com/jesseduffield/lazygit), a dev tool with over a million users around the globe.

Some of my other projects:

- [Lazydocker](https://github.com/jesseduffield/lazydocker) - a terminal-UI for docker
- [Horcrux](https://github.com/jesseduffield/horcrux) - a program that encrypts your file into fragments that can then be recombined to obtain the original file, without the need for a password.
- [_OK?_](https://github.com/jesseduffield/ok) - a joke programming language
- [AI Battlegrounds](https://github.com/jesseduffield/ai-battlegrounds) - A simulator which pits LLM agents against eachother in a 2D environment with weapons, traps, and _blood contracts_.

I also have another blog where I think out loud about psychology and philosophy but I'm too scared to deanonymise that one yet. Let me know if you want that to change!
